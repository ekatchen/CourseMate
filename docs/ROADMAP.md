# Roadmap

## Done (MVP)
- Public tutor directory with course search
- Tutor application form → Supabase
- Student request form → Supabase
- Admin dashboard: approve/reject tutors, manage requests, approve reviews
- Static demo data fallback when no live tutors exist
- How it Works and About pages
- Academic integrity agreement on both forms

## Near-term
- Email notifications: notify admin on new application/request; notify tutor on approval
- Tutor profile page pulls from Supabase (currently reads static demo data)
- Review submission form on tutor profile page
- Protect /admin with password or Supabase Auth

## Medium-term
- Student dashboard: view and manage own requests
- Tutor dashboard: view incoming student contacts
- Verified grade badges (manual admin verification)
- Expand to additional universities beyond Western

## Later / Not yet planned
- In-platform messaging
- Booking and scheduling
- Payments or platform fee
- AI matching
- Full university course directory import
- Mobile app
